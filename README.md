과제소개
미니 도구 모음(Utility Pack) 제작

최정민: 단위 변환기의 기능을 작성, main 화면 틀 구현
main화면에서 1을 입력하면 단위 변환기 기능이 수행됨. 어떤 단위 변환기를 고를 지 이름을 입력하고 숫자를 입력하면 단위가 변경됨.

고효빈: 점수 평균 계산 기능 작성, main 화면 틀 구현
main화면에서 3을 입력하면 점수 평균 계산 기능이 수행됨. 점수를 입력한 후 0을 입력하면 평균 계산이 됨

황지호: 문자열 처리(대/소문자 변환)기능 작성.
 main 화면에서 4를 입력할시 문자열 처리 기능이 실행됨. 대문자로 변환을 원하면 1, 소문자로 변환을 원하면 2를 입력하고 변환하고자하는 단어를 적으면 문자열 처리가 진행됨

김가은: 간단 계산기 기능 작성함. 
main화면에서 2를 입력하면 간단 계산기 기능(사칙연산)이 수행됨. 사용자로부터 숫자 2개와 연산자를 입력받으면 해당 연산자에 따라 두 숫자에 대한 계산이 진행됨.

조현윤: 랜덤 비밀번호 생성기 기능을 작성. 
main화면에서 5를 입력하면 영어대문자 3자리 소문자 5자리 숫자 4자리 특수문자 한자리를 랜덤으로 가져와서 비밀번호를 생성함. 

협업 기록
PR : 총 5개
충돌 : main에서 import sys에서 '#충돌' 텍스트 추가해 랜덤 비밀번호 생성 브랜치와 합병할 때 충돌 발생하게 함. 해결은 '#충돌' 텍스트를 지운 후 import 가져와야할 함수들을 마저 작성함.
